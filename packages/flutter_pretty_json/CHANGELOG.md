## 0.1.0

* Initial release of `flutter_pretty_json`.
* Added `PrettyJson` widget to render JSON as an expandable tree view.
* Added syntax highlighting for keys, strings, numbers, booleans, and null values.
* Added long-press contextual actions: copy value, copy name, and copy property path.
* Added light and dark theme support via `PrettyJsonTheme` (`ThemeExtension`).
